현재 위치 기록하기

현재 위치 정보를 json data 형태로 기록한다.

  • URL

    /:token/position/:email/set/:lat/:lng/:height

  • Method:

    GET

  • URL Params

    Required:

    token=[string]
     email=[string]
     lat=[string]
     lng=[string]
     height=[height]

  • Data Params

    None

  • Success Response:

    • Code: 200 
      Content: { result : "success" }

  • Error Response:

    • Code: 404 NOT FOUND 
      Content: { error : "No user data" }

    OR

    • Code: 401 UNAUTHORIZED 
      Content: { error : "You are unauthorized to make this request." }

  • Sample Call:

      $.ajax({
    url: "http://apis.airpage.org/abcdefg/position/hello@mail.com/set/222.2222/555.55555/900",
    dataType: "json",
    type : "GET",
    success : function(r) {
      console.log(r);
    }
  });
