현재 내위치 기록하기

- URL
/:token/position/:email/set/:lat/:lng/:height

- Method:
GET

URL Params
Required:
token=[string] , 토큰 발급 받기에서 부여받은 토큰
email=[string] , 토큰을 발급받을 때 입력한 이메일
lat=[string] , 드론의 latitude 좌표 (선호좌표계 사용가능)
lng=[string] , 드론의 longtitude 좌표 (선호좌표계 사용가능)
height=[string] , 드론의 고도 (선호단위 사용가능)
Data Params
None

성공시 응답
Code : 200
Content : {result : "success"}

오류시 응답
Code : 404 NOT FOUND
Content : {result : "error" , msg : "No user data" }

호출 예제
$.ajax({
  url :
    "http://apis.airpage.org/apdef/position/hello@mailmail.com/set/123.11/222.33/90",
  dataType : "json",
  type : "GET",
  success : function(r) { console.log(JSON.stringify(r)); }
});
